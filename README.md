## Quick Start

```bash
# Install dependencies
pnpm install

# Start development
pnpm dev

# Build all packages
pnpm build

# Run tests
pnpm test
```

## Project Structure

- `packages/core` - Main UI component library
- `packages/docs` - Documentation website
- `packages/playground` - Example application and component playground

## Development Requirements

- Node.js 16+
- PNPM 8+
- Vue 3
- TypeScript 5+
